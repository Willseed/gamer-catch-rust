安全提醒：service account 與 Gmail OAuth 都只填 JSON 檔名；請勿貼上路徑或 JSON 內容。
這台瀏覽器會自動保留草稿 30 天

草稿會寫入本機儲存空間（Local Storage），包含遊戲名稱、試算表網址或 ID、憑證檔名與通知電子郵件，但不包含 JSON 內容。Local Storage 不是加密保管庫；資料不會上傳，但同一瀏覽器中的網站程式可讀取。若使用共用電腦，完成後請清除草稿。
